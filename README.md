# Serwis-Aukcyjny

W ramach projektu chce utworzyć system umożliwiający wystawianie aukcji oraz licytowanie przedmiotów. Dodatkowo system ma umożliwiać zakładanie kont dla użytkowników oraz umożliwiać ocenę transakcji.
